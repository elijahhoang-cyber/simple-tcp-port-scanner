# Simple Python TCP Port Scanner

## Features

- TCP port scanning
- Detects open/closed ports
- Custom port ranges
- Built using Python sockets

## Technologies Used

- Python 3
- socket module
- TCP networking

## Usage

## Usage

```bash
python3 port_scanner.py
```

Example input:

```text
Input the IP you want to scan: 192.168.1.35
Start port: 1
End port: 100
```
